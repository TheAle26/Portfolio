⚽ Bienvenido a FulboApp

FulboApp es una plataforma liviana y eficiente para organizar tus ligas de fútbol amateur.

¿Qué podés hacer en FulboApp?
  • ✅ Crear y gestionar ligas.
  • ✅ Agregar y editar jugadores con sus estadísticas.
  • ✅ Registrar partidos y puntuar a los jugadores.
  • ✅ Visualizar datos de rendimiento y seguimiento individual.
¿En un futuro!
  • ✅ Ver cachas cercanas.
  • ✅ Ver Niveles de las ligas.
  • ✅ Visualizar datos de rendimiento y seguimiento individual.
¿Cómo fue desarrollada?

Esta aplicación fue desarrollada con Django (Python) y utiliza Bootstrap 5 para el diseño. Corre en una Raspberry Pi Zero 2W, demostrando que se pueden lograr sistemas web funcionales y rápidos incluso con hardware limitado.

Servidor actual: Raspberry Pi Zero 2W
